# WikiCards Task Plan

## Goal
Implement a Next.js web app for WikiCards, a collectible card game based on Wikipedia articles.

## Phases
1. [x] **Phase 1: Setup & Architecture**
   - Initialize Next.js project with Tailwind, TS, and App Router.
   - Set up `planning-with-files` tracking.
   - Design system foundation (colors, typography via `ui-ux-pro-max` skill).

2. [x] **Phase 2: Wikipedia API & Rarity Engine**
   - Create lib functions to fetch random articles.
   - Create lib function to fetch pageviews and calculate rarity.
   - Create `/api/booster` handler that returns 5 cards (guaranteeing >= 1 uncommon+).

3. [x] **Phase 3: UI Components**
   - Create `RarityBadge` component.
   - Create `Card` component with visual styles matching rarity, 3D tilt effect, and holographic overlays.
   - Create `Navbar`.

4. [x] **Phase 4: Booster Opening Experience**
   - Animated sequence to reveal 5 cards (one by one).
   - LocalStorage integration to save drawn cards.

5. [x] **Phase 5: Collection Page**
   - Grid layout to show collected cards.
   - Filters and missing cards styled as silhouettes.
